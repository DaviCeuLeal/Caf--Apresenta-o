# ☕ Café Central Mobile

## Sobre o Projeto

O Café Central Mobile é um aplicativo desenvolvido em React Native com Expo para simular o sistema de uma cafeteria.

O projeto permite que usuários realizem cadastro, login, visualizem o cardápio e enviem mensagens através da página de contato.

## Funcionalidades

* Cadastro de usuários
* Login de usuários
* Visualização do cardápio
* Página sobre a empresa
* Envio de mensagens de contato
* Integração com banco de dados MySQL

## Tecnologias Utilizadas

* React Native
* Expo
* JavaScript
* Node.js
* Express
* MySQL
